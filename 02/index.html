Розмітка заголовків

HTML пропонує шість заголовків різного рівня, що показують відносну важливість розділу, розташованого після заголовка. Так, елемент <h1> (від англ. heading - заголовок) є найбільш важливим заголовком першого рівня, а <h6> служить для позначення заголовка шостого рівня і є найменш значним. За замовчуванням заголовок першого рівня відображається найбільшим шрифтом жирного зображення, заголовки наступного рівня за розміром менше. Елементи <h1>,...,<h6> відносяться до блокових елементів, вони завжди починаються з нового рядка, а після них інші елементи відображаються на наступному рядку. Крім того, перед заголовком та після нього додається порожній простір.

Синтаксис:

<h1>Заголовок 1-го рівня</h1>

<h2>Заголовок 2-го рівня</h2>

<h3>Заголовок 3-го рівня</h3>

<h4>Заголовок 4-го рівня</h4>

<h5>Заголовок 5-го рівня</h5>

<h6>Заголовок 6-го рівня</h6>

Результат:

Заголовок 1-го рівня

Заголовок 2-го рівня

Заголовок 3-го рівня

Заголовок 4-го рівня

Заголовок 5-го рівня
Заголовок 6-го рівня

Теги для форматування тексту

<strong>

Елемент <strong> (від англ. strong - сильний) виділяє важливий текст, відображається жирним зображенням. Важлива пропозиція має велике значення для всієї веб-сторінки, що вказує на щось серйозне або термінове. Важливе слово або словосполучення показує, що воно має більше значення, ніж оточуючий текст. Браузери відображають текст усередині <strong> жирним зображенням.

Синтаксис:

<strong>Текст</strong>

<small>

HTML4 <small> (від англ. small — маленький) зменшує розмір шрифту на одиницю проти звичайним текстом. У HTML4 розмір шрифту вимірюється в умовних одиницях від 1 до 7, середній розмір тексту, який використовується за замовчуванням, прийнятий 3. Таким чином, додавання <small> зменшує текст на одну умовну одиницю. Допускається застосування вкладених елементів <small>, при цьому розмір шрифту буде меншим з кожним вкладеним рівнем, але не може бути меншим, ніж 1. У HTML5 елемент <small> використовується для виносок та коментарів, написаних зазвичай дрібним текстом. До них можуть належати застереження, інформація про ліцензію, авторські права та ін.

Синтаксис:

<small>Текст</small>

<dfn>

Як правило, у документі, коли згадується новий термін, він виділяється курсивом та дається його визначення. При використанні цього терміна надалі він вважається вже відомим читачеві. Елемент <dfn> (від англ. definition - Визначення) застосовується для виділення таких термінів при їх першій появі в тексті.

<abbr>

Елемент <abbr> (від англ. abbreviation - абревіатура) вказує, що послідовність символів є абревіатурою. За допомогою атрибуту title дається розшифровка скорочення, що дозволяє розуміти абревіатуру тим людям, які з нею не знайомі. Крім того, пошукові системи індексують повнотекстовий варіант скорочення, це може використовуватись для підвищення рейтингу документа.

Синтаксис:

Списки

<ul>

Елемент <ul> (від англ. unordered list - невпорядкований список) встановлює маркований список. Кожен пункт списку повинен починатися з <li>.

Синтаксис:

<ul> <li>пункт маркированого списку</li> </ul>

Результат:

  • пункт маркированого списку

<ol>

Елемент <ol> (від англ. ordered list - упорядкований список) встановлює нумерований список. Кожен елемент списку має починатися з <li>. Якщо <ol> застосовує таблицю стилів, то елементи <li> успадковують ці властивості.

Синтаксис:

<ol> <li>елемент нумерованого списка</li> <li>елемент нумерованого списка</li> </ol>

Результат:

  1. елемент нумерованого списка
  2. елемент нумерованого списка

<li>

Елемент <li> (від англ. list item - пункт списку) визначає окремий пункт списку. Зовнішній елемент <ul> або <ol> встановлює тип списку – маркований чи нумерований.

Синтаксис:

<ul> <li>елемент маркированого списка</li> </ul> <ol> <li>елемент нумерованого списка</li> </ol>

Результат:

  • елемент маркированого списка
  1. елемент нумерованого списка

Гіперпосилання

<a>

Елемент <a> (від англ. Anchor - якір) є одним з важливих в HTML і призначений для створення посилань. Для цього необхідно повідомити браузер, що є посиланням, а також вказати адресу документа, на який слід зробити посилання. Як атрибут href використовується адреса документа, на який відбувається перехід.

Синтаксис:

<a href="<адреса>">...</a>

Результат:

адреса

Зображення

<img>

Елемент <img> (від англ. image - зображення) призначений для відображення на веб-сторінці зображень у графічному форматі GIF, JPEG, SVG або PNG. Адреса файлу з зображенням задається через атрибут src. Якщо необхідно, малюнок можна зробити посиланням на інший файл, помістивши <img> у контейнер <a>.

Синтаксис:

<img src="<адрес>" alt="<текст>">
